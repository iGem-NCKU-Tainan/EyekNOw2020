function Inverse_Laplace_Estimator

%INVERSE_LAPLACE_ESTIMATOR provides a close approximation of functions that
%cannot be transformed by inverse Laplave transform function built in
%MATLAB. We utilize inverse fast fourier transform to realize our aims.
%Below are codes demonstrating the method.

Introduction to use of FFT and IFFT

%Just as code listed below;
%x = [1,2,3,4,5];
%sol = fft(x);
%ifft(sol);

Variables that will be used

syms s;

Index that will be used

%Description: when t is larger, the estimation will be more closer to real
%answer.
t = 2000;

Function that needs to be transformed

F(s) = 1/(s+0.003);

Estimation

container = zeros(1,t);
for k = 1:1:t
    container(k) = F(2*pi*1i*(k-1)/t);
end
sol = ifft(container,'Symmetric');

Plot out the estimation and solution

%For estimation
figure;
h = animatedline('color','r','linewidth',1.5);
for j = 1:1:t
    addpoints(h,(j-1),sol(j));
end
hold on;

%For real
syms x;
realsol = ilaplace(F,s,x);
fplot(realsol,[0,t]);
legend('estimation','real');
xlabel('x');
ylabel('f(x)');
title('Estimation and solution of f(x)');
hold off;
